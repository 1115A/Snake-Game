 Installation

 Prerequisites

 Windows OS (for EasyX graphics library)
 C++ compiler (Visual Studio recommended)
 SQLite3 development libraries

 Build Instructions

1. Clone the repository:
bash
git clone https://github.com/yourusername/snakegame.git
cd snakegame


2. Open the project in Visual Studio

3. Ensure EasyX graphics library is installed
    Download from: https://easyx.cn/
    Or install via NuGet in Visual Studio

4. Build the project

 Database Schema

The system uses 5 main tables:

 players: Player information and statistics
 game_records: Individual game session data
 food_records: Food item tracking
 achievements: Player achievement unlocks
 system_config: Game configuration settings

 Usage

1. Run the executable
2. The animated start screen will appear automatically
3. Click "Start Game" to begin
4. Control the snake using:
    W/A/S/D or Arrow Keys for movement
    ESC to exit
    R to restart after game over
5. View statistics and achievements from the game over screen
